# NovaHack

A Mega Hack Pro-style cheat menu for Geometry Dash — 7 floating columns packed with 42 hacks, fully persistent config, and per-hack keybinds that work even when the panel is closed.

## Features

- **MHP-style UI** — 7 independent floating columns (Gameplay, Visual, Physics, Cosmetic, Bypass, Audio, Settings), no unified popup frame
- **42 hacks** — Noclip, Auto Restart, Fast Complete, Freeze Gameplay, Low/High/Moon/Anti Gravity, No Glow, Mirror Mode, Bypass Filter, and many more
- **Per-hack keybinds** — Assign any key to any hack; shortcuts work globally even with the panel closed
- **Custom menu key** — Change the panel open key (default: M) to any key you want
- **Speed control** — Presets (0.25x, 1x, 2x, 5x, 10x) plus a free-form custom speed field
- **FPS cap** — Unlimited or any value you set
- **Freeze Gameplay** — Pauses the game and music instantly without opening the ESC menu; unpause by toggling it off
- **Persistent config** — All hack states, keybinds, speed and FPS are saved automatically across game restarts

## How to use

| Action | Default |
|--------|---------|
| Open/close panel | `M` |
| Toggle a hack | Click its name |
| Assign keybind | Click the `[key]` badge next to a hack, then press any key |
| Remove keybind | Click the badge, then press `ESC` or `Delete` |
| Change menu key | Settings column → Tecla menu → click badge |
